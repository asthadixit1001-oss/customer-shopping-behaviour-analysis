# 📊 Customer Shopping Behaviour Analysis

## 🚀 Project Overview

This project analyzes customer shopping behavior using:

* Python (Data Cleaning & EDA)
* SQL (Advanced Analysis)
* Power BI (Dashboard Visualization)

The goal is to uncover insights related to:

* Customer segmentation
* Revenue drivers
* Purchase patterns
* Subscription behavior

---

## 📁 Dataset

The dataset includes:

* Customer demographics (age, gender)
* Purchase details (category, amount)
* Behavioral data (previous purchases, subscription)
* Marketing factors (discounts, shipping)
* Customer feedback (ratings)

---

## 🧹 Data Cleaning (Python)

* Removed null values
* Fixed data types
* Created age groups (Young Adult, Adult, Middle-Age, Senior)
* Ensured data consistency

---

## 📊 SQL Analysis Insights

* Young Adults generate the highest revenue
* Clothing is the top category
* Discounts increase purchase frequency
* Subscribers spend more than non-subscribers
* Loyal customers drive major revenue

---

## 📈 Dashboard (Power BI)

Key metrics:

* Total Revenue
* Avg Purchase Value (~$59.76)
* Subscription Distribution
* Category Performance

---

## 💡 Business Recommendations

* Target young adults with personalized marketing
* Promote subscriptions for higher lifetime value
* Focus on loyal customers
* Optimize discount strategy
* Invest in top-performing categories

---

## 🏁 Conclusion

This project demonstrates how data can drive business decisions by identifying key customer segments and revenue patterns.
